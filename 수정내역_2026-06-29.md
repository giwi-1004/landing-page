## 2. 섹션 4 — Plan + Loss 통합 (`section-plan.tsx`)

### 플랜 파트

**제목** `3단계면 충분합니다`
- font-size: 20px
- font-weight: 800
- color: `#0F3460`
- margin-bottom: 20px

**3단계 플로우** (기존 흰 박스 제거 → 수평 플로우)
- flexbox, `justify-content: space-between`, `align-items: flex-start`
- 원형 번호: 40×40px, bg `#0F3460`, 흰색 bold 텍스트 (`01`, `02`, `03`)
- 원 사이 점선: `border-top 2px dashed #D6E4F0`
- 단계 라벨 (원 아래): 12px, `#444`, 가운데 정렬
  - 01: 카카오/전화 연락
  - 02: 보험 분석
  - 03: 원하시면 상품 검토

**약속 칩** (기존 구분선 스타일 제거)
- 3개 칩 가로 배치, flex wrap, gap 6px, 가운데 정렬
- 각 칩: bg white, border `1px solid #1D9E75`, border-radius 99px, padding 6px 14px
- font-size 11px, color `#1D9E75`, font-weight 600
- 문구: 가입 권유 없음 · 온라인 상담만 · 결정은 고객님이

### 손실 회피 파트

- 플랜과 **동일 배경** (민트, `landing-section-mint`)
- 별도 배경색 없음 (박스/카드 없음, 텍스트만)

**구분선**
- `border-top 1.5px solid #E0E0E0`, margin 28px 0

**텍스트 (가운데 정렬)**
1. `나중에 하셔도 됩니다.` — 15px, bold, `#1a1a1a`
2. `단, 몸에 이상이 생기면 그때는 늦습니다.` — 13px, `#A32D2D`, margin-top 6px
3. (margin-top 20px) `모아둔 재산이 치료비로 사라지고` — 12px, `#888`
4. `가장의 자리를 잃게 됩니다.` — 16px, font-weight 800, `#A32D2D`, margin-top 4px

**CTA 버튼**
